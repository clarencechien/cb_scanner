## 1. 已定案的研究結論(S0–S10,不要重跑,直接引用)

### 假設鏈判決
原始假設「台股是動能市場,看籌碼就能跟單」:
- 看籌碼跟單 → **否證且已反轉為有害**
- 台股是動能市場 → **2022 後為真,2013–2021 為假**(主導因子會輪替)
- 短期資金面主導 → 為真,但公開資料無可提取 edge

### 三條顯學衰減曲線(全部親手量測,形狀相同)
1. **投信連買**:2013–17 有效(連買5日後20日 +23bps)→ 2022–26 反噬
   (−83bps,t≈−5.6;連買10日 −160bps,t≈−7.1)。死因:全民跟單擁擠。
   附帶發現:法人籌碼殘存資訊集中在**賣方**(投信賣、外資賣的訊號存活,
   因為散戶不跟賣)。
2. **鎖漲停**:PIT 修正後,買得到的漲停 fwd5 平均 +68bps 但**中位數 −48bps、
   勝率 48%**,P10/P90 = −12%/+15.7% → 彩券結構。存活者偏誤+減資假漲停
   曾讓數字灌水四成(+119→+68)。「一字板錯過大肉」是墓碑反光
   (+328→+82,不顯著)。
3. **正2 轉倉紅利**:複製缺口 2015–22 年均 ~+10%(真 edge)→ 2023–25 塌至
   1.7–3.1%,低於股息公平價值 → 超額轉倉已負貢獻。死因:正2 規模推升基差。

### S4 雙重排序(regime 輪替的證據)
| 控制對方後的獨立資訊 | 2013–17 | 2018–21 | 2022–26 |
|---|---|---|---|
| 投信 | 26.8bps | 20.9 | **2.5(死)** |
| 外資 | 14.8 | 17.7 | 8.6(半衰) |
| 動能20日(原始) | 13.7 | 6.8(死) | **43.9(本體)** |

### Beta 容器結論(S8–S10)
- 無腦正2 歷史 CAGR 39.1%/MDD −55%,但引擎拆帳後不可外推
- **A 方案 = 50/50 正2/現金,±10pp 區間再平衡**:22.1%/−29.8%/14 次調整,
  期望 ≈ 0050 + 更好風險形狀 → **已採納為 beta 容器,收工**
- E 方案(加 MA200 條件權重):MDD −21.8 但終值輸 0050、水下時間沒少
  → 保險不是投資,預設不裝
- 再平衡在趨勢市是稅(−9pp/yr vs 永不平衡)——連再平衡都是 regime 賭注
- 無腦正2 水下時間(距高點 −10% 以深)= **43% 的日子**;0050/A/E ≈ 21–22%
- **沒有任何機械規則防得住一夜跳空**;唯一工具是深度價外 TXO put
  (年費 1–2%,屬 Gate 0 決策,未執行)

## 3. 已踩地雷錄(症狀 → 修法,新資料一律先查這五項)

1. **零價格**:FinMind 停牌日 open/close=0 → inf 毒死所有均值。
   修:price<=0→NaN,fwd 算完 replace inf。
2. **分割**:00631L 單日 −95.7%(隱含比 21.75)。修:偵測「ETF/個股單日
   <−25% 且大盤 >−10%」,該日報酬以基準近似或還原。
3. **減資**:單日 >+11.5% 的假「漲停」。修:漲停判定加上限。
4. **存活者偏誤**:用今日流動性選歷史樣本 → 漲停延續灌水四成。
   修:PIT rolling 250 日流動性,`panel_price_full` 已內建。
5. **基準錯位**:分年代事件研究誤用全期基準。修:對照組與事件同時段。
6. 次要:pandas groupby.apply 要 include_groups=False;curves 要 astype(float)
   否則 np.log 爆;matplotlib 缺 CJK 字型(圖內文字用英文或裝 Noto)。

## 4. 方法論決策紀錄(為什麼這樣做,別改)

- **T+1 開盤進場**:法人資料盤後公布;同日報酬只能當幻覺量測。
- **Spearman 橫斷面 IC**:對極值穩健(它在 inf 污染事故中活下來)。
- **事件研究必附**:超額(vs 同時段宇宙)、粗略 t、樣本數、中位數、勝率。
- **成本**:股票來回 0.585%(手續費 0.1425%×2 + 稅 0.3%);ETF 0.385%
  (稅 0.1%);當沖稅減半另計。CB 免證交稅但價差寬 → 用保守滑價假設。
- **分年代切:2013–17 / 2018–21 / 2022–26**,regime 輪替是核心敘事。

## 5. 你的任務:S11 可轉債(台版湯姆熊候選)

### 為什麼是 CB(論證摘要)
台股個股選擇權死市、權證是莊家賭場(發行商=造市者=對手方)、TXO 無橫斷面。
CB = 債底 + 內嵌買權:凸性 ✓ 橫斷面(200–300 檔)✓ 低擁擠(無看盤軟體
排行榜)✓ 資料護城河(條款清洗工程)✓ **制度動機**(發行人有強誘因在
賣回/到期前把股價做上轉換價,否則要掏現金還債)——屬「對手被制度綁定」族,
與已死的資訊差型 edge 不同族。

H-A 發行後正股動能 / H-B 轉換價磁吸 / H-C 賣回日行事曆
