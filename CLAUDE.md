# twchip — 台股結構性 edge 研究(S11: 可轉債)

## 專案脈絡(一分鐘版)
S0–S10 已完成:證明投信跟單(2022 後反噬)、鎖漲停(負中位數彩券)、
正2 轉倉紅利(2023 熄火)三條顯學衰減曲線。結論:公開訊號無 edge,
殘存狩獵場在「對手被制度強迫交易」的角落。S11 檢驗其中最佳候選:
**可轉債(CB)發行人動機假說** —— 發行公司有制度性動機在賣回/到期前
把股價做到轉換價之上(轉換成功 = 債務變股本,免還錢)。

## 核心假說(可證偽)
- H-A(發行動能):CB 發行後 6–18 個月,正股相對大盤有系統性正超額
- H-B(轉換價磁吸):股價接近轉換價下緣時出現異常支撐(回測正股在
  轉換價 -10%~0% 區間的後續報酬 vs 對照)
- H-C(賣回日行事曆):賣回權日前 N 個月的正股超額報酬異常
- 每條假說都要做:分年代穩定性、PIT 宇宙、成本後淨值

## 鐵律(前十步實驗用血換來的,不可違反)
1. **限流**:FinMind 免費額度 ~600 req/hr。任何抓取迴圈 REQ_INTERVAL >= 6.1 秒,
   **絕不平行抓取**。402/429 → sleep 600 秒。抓取一律可續跑(status.json 模式)。
2. **零價格地雷**:FinMind 停牌/無成交日 open/close = 0。任何報酬計算前
   先 `price <= 0 → NaN`,前瞻報酬算完再 `replace([inf,-inf], nan)`。
3. **分割/減資地雷**:單日報酬 < -25% 且大盤正常 → 疑似分割;
   單日 > +11.5% → 疑似減資恢復。偵測、記錄、校正,不可放行。
4. **存活者偏誤**:任何事件研究必須用 PIT(時點正確)宇宙 ——
   用「當時的」rolling 250 日流動性入選,不用今天的名單。
   S6→S7 的教訓:不修這個,數字灌水四成。
5. **可執行性**:進場一律 T+1 開盤價;訊號若涉及盤後公布的資料,
   同日報酬只能當「幻覺量測」,不能當策略報酬。
6. **成本**:股票來回 0.585%,CB 另查(手續費同,免證交稅但價差寬,
   用保守滑價假設並敏感度分析)。
7. **判讀紀律**:統計顯著 != 經濟可行;均值 != 中位數(必附勝率、P10/P90);
   分年代必看(regime 會翻轉);對照組與事件必須同時段。

## 工作流慣例
- 每個假說一個 script:`scripts/s11a_*.py`、`s11b_*.py`...,輸出到 `results/`
- 結果先寫 `results/*.md` 短判讀再迭代,不要只留數字
- 重大發現(正或負)→ 更新本檔的「進度」節
- 影子優先:任何看起來可交易的訊號,先設計每日快照追蹤,不直接下單
