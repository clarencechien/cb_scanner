# S11-D 判讀:下修轉換價事件 —— 對正股否證(又一個賣出訊號)

**判決:下修不是正股的買點。** 下修生效後正股繼續跌:全期(n=196,
降幅 ≥10% 主定義)後 120d 超額中位 −12.2%、勝率 28%、t −5.7,
三年代同向(2013–17 −9.0 / 2018–21 −6.1 / 2022–26 −22.8)。
「公司降低轉換門檻=看好轉換」的多頭敘事,在正股上完全不成立。

事前 60 天中位已 −10.9% → 下修是「股價崩了之後」的被動反應;
事後沒有任何回穩,弱者恆弱。連「下修過頭」(新轉換價讓債直接進入
價內,mny>1.05,n=92)的子集後 120d 中位仍 −10.4%。

## 除權息污染處理(重要方法紀錄)

CBTRN 月表的轉換價變動**混雜除權息調整**:降幅 ≥5% 事件 85% 集中
7–9 月除息季。分離規則:主定義取降幅 ≥10%(真下修通常一刀 10–30%,
除息調整多 <8%);敏感度組「≥10% 且非 7–9 月」(最純,n=36)
後 120d 中位 −10.7%,同向 → 結論不靠污染樣本。

## 機制解讀

下修的受益人是 **CB 持有人**(轉換價值一夜跳升),買單的是股東
(稀釋加大)。正股層面,下修 = 公司自認股價回不去 + 未來稀釋更重,
市場的定價反應是理性的。S11-C 的推論獲得閉環:公司確實用下修
(而非做價)回應賣回壓力——而下修本身再度傷害正股。

**正股層面的 via negativa 補充:發 CB(S11-A)、下修轉換價(S11-D)
都是賣出警訊;CB 相關的正股利多敘事至此全滅。**

## 未殺死

下修對 **CB 價格**(非正股)的事件反應——理論上轉換價值跳升應讓
CB 上漲,若市場反應不足則是 CB 端的事件 edge。需逐日 CB 行情
(月頻太粗,下修當月的跳動被平均掉)。列入 S11-E 之後的待驗佇列。
