# cb_scanner — S11 可轉債假說驗證(台股結構性 edge 研究)

## TL;DR:所以 CB 到底可不可以搞?

**原本的假設(買發 CB 公司的股票)——不可以搞,而且要反著用。**

原始假說是:公司發了可轉債,就有強烈動機把股價做到轉換價之上
(轉換成功=債務變股本、不用還錢),所以正股應該會漲。
實測 2013–2026、含所有已下櫃債的完整樣本,**這個故事是反的**:

1. **公司發 CB 後,股票平均跌給你看**——發行後一年,正股中位數
   輸櫃買指數 14.5 個百分點,只有 31% 的機率贏大盤,13 年來
   三個市場環境都一樣。原因:發債時點是公司挑的(股價高估時發),
   之後是均值回歸+稀釋+套利盤賣壓。
2. **股價跌到轉換價附近不會有人「護盤」**——所謂磁吸支撐不存在。
3. **賣回日之前公司也不會拉股價**——公司缺錢時的選擇是「下修
   轉換價」(改遊戲規則,一紙公告),不是花真錢買股票。
4. **下修轉換價本身又是一個賣出訊號**——下修後 120 天股票中位數
   再輸大盤 12.2 個百分點。

一句話:**看到「發 CB」或「下修轉換價」,不要買,持有的要警戒。**
這兩條反向規則是本次研究最有價值的產出(零成本、14 年穩定)。

**意外的活口(買 CB 本身,不是買股票)——統計上可以,執行上很勉強,
先觀察不下單。** 低轉換溢價(<15%)、價格貼近面額(95–115)的 CB,
因為「債底擋跌、轉股權吃漲」的不對稱,月均毛報酬 +1.25%、14 年
每年都是正的。但:CB 市場很薄(一半的債整天沒成交),買賣價差
就能吃掉大半利潤(滑價 1% 來回→淨年化約 7%;2% 就歸零),
整個策略容量只有幾千萬台幣,純散戶尺度。按本專案鐵律:
**先跑 6 個月影子模式(只記錄不下單)驗證真的買得到,才談上線。**

## 深層解讀:五案拼出的一張圖

五個判決疊起來,浮現的不是五個獨立事實,是一條水管:
**CB 市場是從「發行公司股東」流向「債券持有人(+知情內部人)」的
財富移轉管道。** H-A 的死和 S11-E 的活是同一筆錢的兩本帳——
股東出的錢,就是 CB 持有人收的錢。

**1. 發 CB 的公司不必然爛,但池子被三重機制負向篩選過:**
- *信用約束*:能便宜跟銀行借的公司就直接跟銀行借了;留在 CB 池的
  先天信用偏弱一層(0% 票面是它們最便宜的無擔保融資)。
- *估值擇時*(Myers-Majluf 教科書劇本):發行時點是管理層挑的,
  他們什麼時候願意賣「未來用股票還債」的權利?自認股價貴的時候。
  CB 發行本質是一次知情者賣出——發行後一年 −14.5pp 不是異常,
  是逆選擇理論在台股的實測簽名。
- *會計幻覺*:0% 票面「不用付利息」,送出去的選擇權價值不進損益表
  → 公司系統性把內嵌選擇權定價過便宜(慷慨不是慈善,是會計盲點)。

「弱勢池 60d 中位 −3~−4%」就是三重篩選的總和量測。
(注意:83% 下櫃 ≠ 死亡率——CB 下櫃多數是轉換完畢或到期贖回,
它是回測偏誤的來源,不是倒閉統計。)

**2. H-C 全錯的方法論教訓:我們把對手的行動空間搞錯了。**
「公司被制度強迫避免現金贖回」是真的,但理性代理人走最便宜的路:
下修轉換價(一紙公告,股東買單)永遠比護盤(真金白銀)便宜。
制度強迫假說要成立,必須先窮舉對手所有洩壓閥,再確認你賭的那個
最便宜。下修的財務本質是把股東的錢轉給債券持有人——所以下修後
正股再跌 12.2pp(S11-D)完全自洽:壞消息確認 + 價值移轉公告。

**3. 研究分類學:這次殺的不是衰減曲線,是從未活過的神話。**
投信連買、鎖漲停、正2 紅利是「曾經活過、被擁擠吃死」的 edge;
「發 CB 公司會把股價做上去」則是 2013 年就反、十四年一直反的
民間故事——它流傳不是因為曾經有效,而是敘事動聽且沒人測過。

**4. 樣本外對帳:台達 EB × 泰達電(誠實記錄,含打臉)。**
2025/1 台達子公司發 US$525M 零票面可交換債(EB),交換標的是
手上持有的泰達電(DET.BK)股票,交換價 THB 187.6 = 參考價(~134)
+40%。這是同一條邏輯的世界一流版本:知情者(母公司)對富價資產
(子公司股票)的擇時賣出,還以 0% 票面收走權利金——**與中小公司
發 CB 賣自家高估股票,動作同構**。
樣本外驗證(2026-07 查核):DET 先在 2025 年中崩到 92(−31%,
前半年「知情賣出」方向神準)……然後隨 AI 資本支出潮飆到 325
(2026-06-30 收盤,對參考價 **+142%**,交換權深度價內)。
**方向預言在 18 個月尺度上錯了,認。** 但注意輸贏歸屬:EB 持有人
零息買 100,崩盤時債底+賣回權托著,反彈時平價衝到面額的 ~173%
——**發行人賣便宜凸性、買方收割,S11-E 那枚硬幣的完美單例**
(凸性買方的夢幻路徑恰恰是「先崩後飆」)。單一案例的方向故事
會輸給 regime,池子的統計和凸性的不對稱才是可依賴的部分。
(技術注記:海外 EB 不在櫃買國內 CB 宇宙裡,未污染本研究樣本;
未來擴充宇宙時 EB 與海外 CB 是不同物種,不可混池。)

**5. 對 S11-E 影子期的加掛要求(承接弱勢池邏輯):**
債底只在不違約時存在,而這個池子信用先天偏弱。影子期除了驗證
成交可行性,同步記錄每檔的淨值比、CB 餘額/市值、是否連續發行人
(連續發 CB 是三重篩選的極端樣本)——六個月後多一層
「哪些債底是真地板、哪些是活板門」的實證。

> 一句話:**發 CB 不是公司有問題的證明,是知情者認為股票貴了的
> 證明——而台達只是把同一句話,用世界一流的財務工程又說了一遍。
> (泰達電後來翻倍是給敘事的當頭棒喝:知情者也會賣飛,
> 但賣便宜凸性的錢,終究是被債券持有人賺走的。)**

## 五案判決(給要看數字的人,2026-07-07 收官)

| 假說 | 判決 | 關鍵數字 |
|---|---|---|
| H-A 發行後正股動能 | **否證+反轉** | 發行後 250d 正股中位輸櫃買指數 14.5pp、勝率 31%;2013–17 / 2018–21 / 2022–26 三年代同向(−14.5 / −14.3 / −15.0);vs 等權 CB 發行人同儕仍 −18.2pp |
| H-B 轉換價磁吸 | **否證** | 35,764 債·月:磁吸帶(股價/轉換價 0.90–1.00)後續報酬與鄰桶無異,型態是動能不是支撐 |
| H-C 賣回日行事曆 | **否證** | 賣回前 120d「做價期」超額中位 −6.7%、勝率 35%(n=452);價外子集也無效果。公司的洩壓閥是下修轉換價,不是做股價 |
| S11-D 下修事件(正股) | **否證** | 下修 ≥10% 後 120d 超額中位 −12.2%、勝率 28%,三年代同向;「下修過頭到價內」也救不回 |
| S11-E CB 本體凸性 | **未否證→影子** | 溢價<15% 且價 95–115:月均毛 +1.25%、14 年逐年全正、P10/P90 −3.4%/+6.2%(右偏 1.8×);滑價 1% 來回淨年化 ~7%、2% 即死;容量 ~3–6 千萬台幣 |

**可用產出(via negativa 第五、六條)**:公司發 CB 後一年內正股不做多、
既有持股視為出場警訊;公司下修轉換價 ≥10% 同樣是賣出警訊。

**附帶發現**:CB 發行人是慢性弱勢池 —— 存續期任一時點、任一價位,
60d 超額中位 −3~−4%(vs 櫃買指數)。做多任何「CB 發行人正股」策略,
起跑線在水面下。

**兩個新地雷**:(1) 基準指數 regime —— 2025-06→2026-07 TAIEX +112%,
中小型股嚴重落後,單一大盤基準會把超額全壓負(已改三基準並列);
(2) CB 存活者偏誤加倍 —— 2013–24 首見的債 83% 已下櫃,用「存續名單」
做的 CB 回測直接作廢(本研究用逐月歷史行情檔重建 PIT 宇宙)。

**沒殺死的**:S11-E 執行面(T+1 可執行性需逐日 CB 行情)、下修事件的
CB 端反應、打新(外部資料)。正股方向已全數驗畢,不再回頭。
